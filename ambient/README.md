Quick start:

```sh
ambient new testy
cd testy
ambient run
```

For more details, go to the [Ambient documentation](https://ambientrun.github.io/Ambient/introduction.html)

# Installing on Linux

To run Ambient on Linux you need to have libasound2 installed (i.e. `apt-get install libasound2`).
